MAX Fake for iOS

Минимальная версия iOS: 18.5
Установка: через Sideloadly или через собственный корпоративный сертификат
Совместимость: адаптивно под все размеры экранов (от iPhone SE до Pro Max)

Установка через Sideloadly:

1. Скачай ZIP ниже
2. Распакуй → получишь `MAXFake.ipa`
3. Открой Sideloadly
4. Перетащи `.ipa` в окно программы
5. Введи Apple ID → Start
6. После установки: Настройки → Основные → Профили → доверие сертификату

Установка через корпоративный сертификат:

1. Используй свой корпоративный сертификатподписывай и подпиши .ipa через него.
2. Установи приложение на устройство.
